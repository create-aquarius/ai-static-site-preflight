# Roadmap

## v0.1: Browser preflight checker

- [x] Local-only browser interface
- [x] Metadata, accessibility, path, and static-hosting checks
- [x] Downloadable Markdown report
- [x] Dependency-free automated tests

## v0.2: Better reporting

- [ ] Show the source snippet for each finding
- [ ] Add severity filtering
- [ ] Add rule documentation with before-and-after examples
- [ ] Add Japanese interface text

## v0.3: Folder and command-line checks

- [ ] Add a CLI for checking HTML files in a local folder
- [ ] Detect missing local images, stylesheets, and scripts
- [ ] Add JSON output for automation

## v0.4: Maintainer automation

- [ ] Publish a reusable GitHub Action
- [ ] Add pull request annotations
- [ ] Add release notes and versioned rule documentation

## Scope boundary

The project will remain focused on technical checks for small static sites. It
will not add marketing scoring, sales-copy prompts, or private business
playbooks.
