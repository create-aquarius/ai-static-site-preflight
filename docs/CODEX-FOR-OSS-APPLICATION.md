## Why does this repository qualify?

`ai-static-site-preflight` is a local-first open-source checker for
AI-generated static HTML. It helps people catch common metadata,
accessibility, path, and GitHub Pages compatibility issues before publishing.
The project is designed as a practical safety layer for small static sites.

## How will you use API credits for your project?

## What maintenance tasks do you handle?

## What future improvements are planned?

Planned improvements include clearer source-level findings, severity filters,
rule documentation, a CLI for folder-level checks, missing-asset detection,
JSON output for automation, and a reusable GitHub Action with pull request
annotations.

## Anything else we should know?

The project intentionally keeps HTML analysis local in the browser and avoids
uploading user content. It focuses on practical, explainable checks for people
publishing small static sites, including pages created with AI assistance.
